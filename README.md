# Time-roulette

## 1.front end
##### 安装工具
```
    npm install vue
    npm install element-plus --save-dev
    npm install typescript --save-dev
    npm install vue @types/vue --save-dev
    npm install @element-plus/icons-vue
    npm install vue-router@4
    npm install vuex 
    npm install axios
```

##### 启动
```
    cd ./frontEnd/time_roulette/
    npm install
    npm run -dev
```
